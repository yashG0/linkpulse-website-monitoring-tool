=========================================================

---------------------------------------------------------

using Go, SQLite/PostgreSQL, goroutines, and REST APIs.

=========================================================
PROJECT STRUCTURE
=========================================================

linkpulse/
│
├── cmd/
│   └── api/
│       └── main.go              # Application entry point
│
├── internal/
│   ├── checker/
│   │   ├── checker.go           # Check a single URL
│   │   └── worker.go            # Background scheduler & workers
│   │
│   ├── config/
│   │   └── config.go            # Load configuration from environment
│   │
│   ├── db/
│   │   ├── db.go                # Database connection & initialization
│   │   ├── monitors.go          # Monitor database queries
│   │   └── results.go           # Check result database queries
│   │
│   ├── handlers/
│   │   ├── monitors.go          # HTTP handlers
│   │   └── router.go            # Route registration
│   │
│   ├── models/
│   │   ├── monitor.go           # Monitor struct
│   │   └── result.go            # CheckResult struct
│   │
│   └── server/
│       └── server.go            # HTTP server setup
│
├── .env
├── .gitignore
├── Dockerfile
├── go.mod
├── go.sum
├── README.md
└── plan.txt
